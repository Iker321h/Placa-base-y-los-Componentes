BIOS

¿Qué es la BIOS?

La BIOS (siglas en inglés de Basic Input/Output System o Sistema Básico de Entrada/Salida) es un software de bajo nivel almacenado en un chip de la placa base. Es responsable de iniciar el hardware del sistema y preparar el entorno para que el sistema operativo pueda arrancar correctamente.

Funciones principales de la BIOS

  • Inicialización del hardware: Detecta y verifica que los componentes esenciales como la memoria RAM, CPU, teclado, discos duros, y tarjetas gráficas estén conectados y funcionando.
  • POST (Power-On Self Test): Es una prueba automática que se realiza al encender el ordenador para asegurarse de que el hardware funciona correctamente.
  • Cargar el gestor de arranque: Una vez finalizado el POST, la BIOS busca un dispositivo de arranque (como un disco duro o unidad USB) para cargar el sistema operativo.
  • Configuración del sistema: Permite al usuario modificar parámetros como el orden de arranque, la hora del sistema, la velocidad del ventilador, etc., mediante una interfaz conocida como Setup.

Tipos de BIOS

  • BIOS tradicional: Sistema de texto simple con soporte limitado para hardware moderno, utilizado hasta aproximadamente 2010.
  • UEFI (Unified Extensible Firmware Interface): Reemplazo moderno de la BIOS, con interfaz gráfica, soporte para discos de más de 2 TB, arranque seguro, y mayor rapidez en el inicio.

Diferencias entre BIOS y UEFI

BIOS UEFI
Interfaz de texto simple Interfaz gráfica y con soporte para ratón
Soporte para discos de hasta 2 TB Soporte para discos de más de 2 TB (GPT)
Arranque más lento Arranque más rápido y seguro
No soporta Secure Boot Incluye Secure Boot para proteger el arranque

¿Dónde se encuentra la BIOS?

La BIOS está almacenada en un chip ROM o Flash ROM en la placa base. Este chip mantiene su contenido incluso cuando el equipo está apagado, gracias a que no necesita energía continua para retener la información.

¿Qué es el Setup de la BIOS?

Es una interfaz accesible al encender el equipo (usualmente presionando teclas como Del, F2 o Esc) que permite configurar parámetros del hardware. Desde allí puedes:

  • Elegir el dispositivo de arranque (USB, SSD, CD, etc.)
  • Habilitar o deshabilitar funciones del procesador o dispositivos integrados
  • Supervisar la temperatura y voltajes del sistema
  • Realizar overclocking en ciertos modelos

Importancia de la BIOS

Sin la BIOS (o UEFI), el sistema no podría arrancar ni detectar los componentes esenciales. También actúa como una capa de seguridad, permitiendo proteger el acceso al equipo mediante contraseñas o activar el "arranque seguro" que evita la ejecución de software malicioso durante el encendido.
